Polling System API

This api is used to create the questions and options to voting

  • CREATE A QUESTION
  • DELETE A QUESTION
  • VIEW A QUESTION
  • CREATE OPTION
  • DELETE OPTION
  • ADDING VOTE

GET QUESTION

POST(http://localhost:8000/questions/:id)

CREATE QUESTION

POST(http://localhost:8000/questions/create)

DELETE QUESTION

POST(http://localhost:8000/questions/:id/delete)

CREATE OPTION

POST(http://localhost:8000/questions/:id/options/create)

DELETE OPTION

POST(http://localhost:8000/options/:id/delete)

ADD VOTE TO AN OPTION

GET(http://localhost:8000/options/:id/add_vote)
